# IPL Auction UI Design System

## Overview
This design system defines the visual language for the IPL Auction interface. The goal is to maintain a premium, futuristic, and game-like experience across all pages.

---

## 1. Color System

### Base Colors
--bg-primary: #0A0F12;
--bg-secondary: #0F1A20;
--bg-panel: rgba(15, 26, 32, 0.85);

### Accent Colors
--accent-gold: #D4AF37;
--accent-gold-light: #E6C766;
--accent-gold-dark: #A88A2A;

### Neutral Colors
--text-primary: #FFFFFF;
--text-secondary: #B0BEC5;
--text-muted: #6B7C85;

### Supporting Colors
--accent-blue: #12303A;
--accent-teal: #0F2A33;

---

## 2. Background System
- Radial light rays
- Grid overlay
- Angular shapes
- Gradient blending

Example:
background: radial-gradient(circle at 80% 50%, rgba(255,255,255,0.05), transparent),
            linear-gradient(135deg, #0A0F12, #0F1A20);

---

## 3. Layout System
Left: Visual (logo, team, selector)
Right: Functional (title, forms, actions)

---

## 4. Components

### Buttons
- Angular edges
- Metallic gradient
- No rounded corners

### Inputs
- Dark background
- Thin border
- Glow on focus

### Panels
- Semi-transparent
- Subtle border
- Blur effect

---

## 5. Typography
- Bold sans-serif
- Gold for headings
- White for labels
- Gray for secondary text

---

## 6. Effects
- Hover: glow + slight scale
- Focus: gold glow
- Transition: 0.2s ease-in-out

---

## 7. Principles
- High contrast
- Minimal colors
- Premium spacing
- Game-like interactions

---

## 8. Avoid
- Rounded UI
- Bright random colors
- Flat design
- Overuse of gold

---

## Summary
Futuristic IPL auction arena with premium esports UI aesthetics.
